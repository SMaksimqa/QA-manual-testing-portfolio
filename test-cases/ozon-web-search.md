## Фильтр по цене поиска Ozon (web)
- ID : TC-003
- Тип теста: Функциональный (позитивный)
- Дата: 01.02.2026

**Окружение**:
- Платформа: web
- Браузер: Chrome
- ОС: Windows 11

**Предусловия**: 
- Открыт сайт Ozon (web)
- Пользователь на странице поиска  (запрос "ноутбук")

**Шаги**:
1. Установить фильтр цены: от 50 000 до 100 000 ₽  
2. Применить фильтр
3. Проскролить результаты

**Ожидаемый результат**: 
- Все результаты в диапазоне 50.000-100.000 ₽
- Нет ноутбуков дешевле 50.000 и дороже 100.000 ₽

**Актуальный результат**: 
- Все результаты в диапазоне 50.000-100.000 ₽
- Товары за пределами диапазона не отображаются

**Статус**: Passed
