# Python — Создание файлов по номеру 

**Задача**  
Написать функцию, которая принимает число N и создаёт в текущей папке N пустых текстовых файлов:  
1.txt  
2.txt  
...  
N.txt

**Что сделано**  
- Функция create_files(n) с проверкой входных данных  
- Обработка ошибок ввода (ValueError)  
- Docstring с описанием  
- Вывод названий созданных файлов

**Как запустить**  
1. python create_files.py  
2. Введите число N (например 5)  
3. В папке появятся файлы 1.txt – 5.txt

**Навыки**  
- Функции с аргументами  
- Работа с файлами (open/w)  
- Обработка исключений (try/except)  
- Проверка входных данных  
- Форматирование строк (f-strings)
