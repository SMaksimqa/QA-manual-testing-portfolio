# SQL Examples

Примеры запросов из курса (1.5 месяца изучения).  
Фокус на задачах QA: выборка данных, агрегация, проверка дубликатов, вставка/обновление.

**Что внутри**  
- SELECT с JOIN и фильтрами  
- GROUP BY + HAVING + агрегаты (COUNT, SUM, AVG)  
- Поиск дубликатов / NULL  
- INSERT / UPDATE / DELETE

**Как использовать**  
Копировать в pgAdmin / DBeaver / MySQL Workbench / SQLite

**Навыки**  
JOIN (INNER/LEFT), GROUP BY/HAVING, подзапросы, работа с NULL, DML-запросы
